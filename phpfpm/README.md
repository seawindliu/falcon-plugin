open-falcon php-fpm监控脚本
===========================

安装使用
--------
nginx.conf配置
      location /status {
          #fastcgi_pass 127.0.0.1:9000;
          fastcgi_pass unix:/tmp/php-cgi.sock;
          include fastcgi.conf;
       }

php-fpm设置`pm.status_path=/status`，与脚本中设置相应，并填上php-fpm监控地址或sock文件。

加入crontab中定时执行。
